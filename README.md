# wise-ui

A collection of animated, production-ready React components built with Next.js, Tailwind CSS, and Motion. Installable via the [shadcn registry](https://ui.shadcn.com/docs/registry).

## Components

| Component | Description |
|-----------|-------------|
| **Animated Tabs** | Tabs with smooth animated indicator transitions |
| **Border Beam** | Card with an animated beam traveling along its border |
| **Data Table** | Animated data table with sorting, search, selection, inline editing, column resizing, and pagination |
| **Dock** | macOS-style dock with magnification effect on hover |
| **Dot Matrix** | Animated dot matrix background pattern |
| **Editable Text** | Inline editable text with animated border reveal and shimmer effect |
| **Gradient Sidebar** | Sidebar with animated gradient orb background and collapsible sections |
| **Kanban Board** | Drag-and-drop kanban board with collapsible columns and animated card transitions |
| **Magnetic Button** | Button that magnetically attracts toward the cursor on hover |
| **Marquee** | Infinite scrolling marquee with configurable speed and direction |
| **Metallic Lava** | Animated metallic lava background using WebGL shaders |
| **Moving Gradient Button** | Button with spinning gradient border and flowing gradient text on hover |
| **Moving Gradient Text** | Text with a continuously flowing gradient animation |
| **Nebula** | Animated nebula cloud background using WebGL shaders |
| **Pulse Button** | Button with a pulsing glow animation |
| **Reorderable List** | Drag-and-drop sortable list with smooth animations |
| **Ripple** | Ripple effect animation on click |
| **Scramble Text** | Text that scrambles through random characters before resolving |
| **Spotlight Card** | Card with a spotlight effect that follows the cursor |
| **Tilt Card** | Card that tilts toward the cursor with a 3D perspective effect |

## Installation

Install any component directly using the shadcn CLI:

```bash
npx shadcn@latest add https://wise-ui.com/r/animated-tabs
```

Replace `animated-tabs` with any component name from the table above (use kebab-case).

## Tech Stack

- [Next.js](https://nextjs.org) 16
- [React](https://react.dev) 19
- [Tailwind CSS](https://tailwindcss.com) 4
- [Motion](https://motion.dev) (Framer Motion)
- [Radix UI](https://www.radix-ui.com)
- [dnd-kit](https://dndkit.com) for drag-and-drop

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the docs site.
